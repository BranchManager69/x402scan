## Overview

Facilitators are the entities that process x402 transactions. They are responsible for validating and forwarding transactions to the network.

## How We Use Facilitators

Currently, the only way to indentify "x402 transactions" is to query `Transfer(address,address,uint256)` events submitted by the facilitators.

## Adding Facilitators

If you run a facilitator and want to be added to the list, please open a PR to add your facilitator to the [facilitators](https://github.com/Merit-Systems/x402scan/blob/main/src/lib/facilitators.ts) file.
